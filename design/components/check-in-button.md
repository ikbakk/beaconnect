# Check-in Button

## Label

```text
I'm Okay
```

## Purpose

A one-tap expression of presence.

## States

- idle
- pressed
- sending
- success
- cooldown
- disabled

## Feedback

- gentle compression
- ripple
- haptic
- toast confirmation

## Success Copy

```text
Sarah now knows you're around.
```

## Cooldown Copy

```text
You recently checked in.

You can check in again in a few minutes.
```

## Anti-patterns

No:

- confetti
- checkmark celebration
- guilt copy
- streaks
- pressure
