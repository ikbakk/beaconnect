# Partner Card Component

## Purpose

The Partner Card answers:

- who
- how they are
- how fresh the information is

## Structure

```text
Name + Pair Symbol
Status Sentence
Freshness Sentence
```

## Normal Example

```text
Sarah ⭐

Everything looks normal.

Updated 2 minutes ago at Home.
```

## No New Updates Example

```text
Sarah ⭐

No new updates yet.

Last updated 47 minutes ago near Home.
```

## Live Example

```text
Sarah ⭐

Sharing live.

18 minutes remaining.
```

## Design Rule

Never add technical details to the card.

If information does not reassure, it does not belong.
