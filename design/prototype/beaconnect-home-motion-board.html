Motion behaviors

Quiet motion that teaches cause and effect.

Every motion cue is short, asymmetric, and purpose-bound. Nothing loops for delight alone. The user should only notice movement when it helps explain what just changed.

I’m Okay

I’m Okay press

The primary action compresses by about 1.5%, sends one contained ripple, then returns to rest. The rest of the screen stays still so the action feels calm and deliberate.

  • Press: 140ms in, 180ms out
  • Haptic: soft confirmation only
  • No bounce, no pulse, no celebratory state

Secondary actions open as sheets

Request Check-in and Start Live rise from the bottom as one clear next step. The movement comes from below because it feels additive, not interruptive.

  • Sheet enter: 220ms ease-out
  • Sheet exit: 140ms ease-out
  • Background dims slightly, never dramatically
Current place Map remains secondary and settles behind the main story.

State changes settle into place

When the partner state changes, the card cross-fades its wording and the map preview subtly re-centers underneath. The screen should feel updated, not reloaded.

  • Copy swap: 180ms fade/translate
  • Map settle: 260ms ease-out drift
  • No spinner, no technical language
