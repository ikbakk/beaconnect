# Color Roles

Beaconnect uses role-based colors, not decorative branding.

## Roles

- Surface
- Surface Variant
- Primary
- Secondary
- Outline
- Success / Normal
- Informational
- Recommendation
- Critical

## Principles

- Use Material You dynamic color where available.
- Avoid highly saturated colors.
- Do not use red unless action or state is genuinely urgent.
- Delayed/no-new-updates should feel informational, not alarming.
- Color never carries meaning alone.

## Avoid

- neon gradients
- glass effects
- heavy brand color domination
- dashboard-style color coding
