# Design Tokens

## Grid

Use 8pt grid with 4pt micro-grid.

Common values:

```text
4
8
12
16
24
32
48
64
```

## Rhythm

### Tight

Inside components.

### Comfortable

Between related sections.

### Spacious

Between unrelated sections.

## Elevation

Only three levels:

| Level | Usage |
|---|---|
| Flat | most surfaces |
| Raised | Partner Card |
| Overlay | bottom sheets / dialogs |

## Motion Categories

- state
- expand
- collapse
- feedback
- navigate

## Radius Categories

- Small: chips / small controls
- Medium: cards
- Large: buttons / bottom sheets

Exact values can be finalized during implementation, but do not create random radii.
