# Design Review Framework

Every screen and feature must pass these tests.

## 1. Product Test

Should this exist?

If no, delete it.

## 2. Behavior Test

What behavior does this screen encourage?

Prefer:

- communication
- reassurance
- presence
- trust

Avoid:

- investigation
- obsession
- suspicion
- repeated checking

## 3. Calm Test

Ask:

- Does this reduce anxiety?
- Could this accidentally increase anxiety?
- Does this pressure the user?
- Does it assume intent?

## 4. Human Language Test

Could a non-technical user understand every sentence?

## 5. Grandma Test

Could someone without technical knowledge understand the screen without explanation?

## 6. Three Second Test

Can users answer:

- Are they okay?
- When was the last update?
- Do I need to do anything?

within three seconds?

## 7. Android Test

Does this feel native to Android?

## 8. Beaconnect Test

If the logo were removed, would this still feel like Beaconnect?

## 9. Gravity Test

What is the user's attention orbiting around?

For Beaconnect, the answer should be:

- partner
- relationship
- reassurance

Not:

- button
- map
- logo
- navigation
