# Screen States

## Partner Card States

### 1. Normal

```text
Sarah

Everything looks normal.

Updated 2 minutes ago at Home.
```

### 2. Live Sharing

```text
Sarah

Sharing live.

Going home.

18 minutes remaining.
```

### 3. Sharing Paused

```text
Sarah

Sharing is currently paused.

Last update was today at 14:32.
```

### 4. No New Updates

```text
Sarah

No new updates yet.

Last updated 47 minutes ago near Home.
```

### 5. Permission Missing

```text
Sharing is not fully available yet.

Background sharing works best when permission is enabled.
```

### 6. No Partner

```text
Welcome.

Beaconnect works best when both people choose to share.

Pair with someone to get started.
```

### 7. Shared Connection Ended

```text
This shared connection has ended.

You can start a new shared connection anytime.
```

## State Rule

Every state should preserve the same layout.

Only meaning changes.

Users should not need to relearn the interface.
