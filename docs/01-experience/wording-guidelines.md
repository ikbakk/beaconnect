# Wording Guidelines

## Core Rule

Speak in relationship terms, not system terms.

## Prefer / Avoid

| Avoid | Prefer |
|---|---|
| Track | Share |
| Tracking | Sharing |
| Live Location | Live Sharing |
| GPS | Update / Current place |
| Coordinates | Place |
| Last seen | Last updated |
| Offline | No new updates yet |
| Error | Something did not go as expected |
| Refresh | Request Check-in |
| Monitoring | Staying connected |
| Watch | Check in |
| Permission denied | Permission was disabled |
| Sync failed | We will try again automatically |
| Cache | Most recent update |

## Tone

Use:

- calm
- short
- direct
- kind
- neutral

Avoid:

- blame
- suspicion
- pressure
- panic
- technical explanation

## Examples

Good:

```text
Showing your most recent update.
We'll refresh when possible.
```

Bad:

```text
Network sync failed.
Retry?
```

Good:

```text
No new updates yet.
```

Bad:

```text
Partner offline.
```

Good:

```text
Sharing is currently paused.
```

Bad:

```text
Partner stopped tracking.
```
