# My Beacon v1.0

Status: Frozen

## Purpose

Personalize Beaconnect without changing the shared trust model.

## Scope

```text
Check-in Messages
Pair Symbol
Widget
Quiet Hours
Appearance
Accessibility
```

## Rule

If it changes only your experience, it belongs in My Beacon.

If it changes the relationship or shared trust, it belongs in Trust Center.

## Pattern

```text
Preference
↓
Current choice
↓
Customize
```

## Example

```text
Check-in Messages

Personalize the short messages used when you check in.

Random from 6 messages

Customize →
```

## Non-Scope

My Beacon does not contain:

- relationship controls
- trust controls
- history deletion
- privacy policy details
- diagnostics
- developer options

## Principle

My Beacon makes Beaconnect yours without changing what Beaconnect promises to both people.
