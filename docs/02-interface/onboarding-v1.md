# Onboarding v1.0

Status: Frozen

## Purpose

Onboarding teaches expectations, not features.

## Flow

### Screen 1 — Welcome

```text
Built for reassurance,
not surveillance.

Beaconnect helps two people quietly stay connected through mutual sharing and simple check-ins.

[Get Started]
```

### Screen 2 — Account

Fast.

Display name required.

No profile photo in v1.

### Screen 3 — Pairing

```text
Beaconnect only works when both people agree.

Connections are always mutual and can be ended at any time.
```

### Screen 4 — Permissions

Pattern:

```text
Why
↓
Status
↓
Recommendation
↓
Action
```

### Screen 5 — Connected

```text
You're connected.

Try your first check-in.
```

Action:

```text
I'm Okay
```

### Screen 6 — Success

```text
Sarah now knows you're around.

You're ready to quietly stay connected.
```

## Skip

Pairing can be skipped.

Home adapts:

```text
Beaconnect works best when you're connected with someone.

Pair when you're ready.
```

## Removed

No:

- tutorial carousel
- marketing slides
- feature checklist
- confetti
- mascot
- premium upsell

## Principle

First success is not installation complete.

First success is:

> Someone I care about now knows I am around.
