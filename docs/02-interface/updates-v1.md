# Updates v1.0

Status: Frozen

## Purpose

Updates answers:

> What has changed since I last looked?

## Not

Updates is not:

- system log
- raw GPS timeline
- chat history
- infinite feed

It is a journal of meaningful moments.

## Structure

```text
Today
  Morning
  Afternoon
  Evening

Yesterday

2 Days Ago

3 Days Ago

Monthly Summary entry
```

## Time Language

Prefer:

- Morning
- Afternoon
- Evening
- Late Night

Exact timestamps live in detail view.

## Grouping

Events within 1–5 minutes may become one story.

Rule:

> One moment, one story.

## Event Types

Include:

- check-in
- reaction
- live sharing started
- live sharing ended
- saved place arrival/leave
- battery critical threshold if meaningful
- pairing/trust event

Exclude:

- GPS update
- sync complete
- cache update
- app opened
- widget refreshed
- background task completed

## Details

Tap event to open bottom sheet.

Detail may show:

- exact timestamp
- place
- reaction
- Open in Maps

## Ending

Every day has a gentle ending.

Example:

```text
That's everything for today.

We'll keep you updated when something meaningful happens.
```

## Principle

Completion over engagement.
