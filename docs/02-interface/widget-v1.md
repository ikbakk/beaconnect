# Widget v1.0

Status: Frozen

## Purpose

The widget is a reassurance glance.

It is not a miniature app.

## Size

Reference size:

```text
4×2
```

No other sizes in v1.

## Layout

```text
Sarah

Everything looks normal.

Updated 2 min ago at Home.

[ I'm Okay ]
```

## Visual Hierarchy

Always:

```text
Partner
↓
Status
↓
Freshness
↓
Action
```

## Primary Action

Only one:

```text
I'm Okay
```

## Motion

When pressing I'm Okay:

1. button gently compresses
2. Material ripple
3. small haptic
4. button returns
5. confirmation toast

No bounce.

No glow.

No animation loops.

## No Map

Maps are excluded from v1 widget.

Reason:

The widget is for reassurance, not investigation.

## Excluded

- map
- timeline
- multiple buttons
- battery percentage unless critical
- navigation shortcuts
- decorative animation
- partner photo

## Principle

If users need more information, they open Beaconnect.
