# ADR-000 — Product Promises

Status: Frozen

## Decision

All architecture must protect Beaconnect's product promises.

## Promises

1. Dependable
2. Calm
3. Graceful degradation
4. Battery respect
5. Honest freshness
6. Offline tolerance
7. Predictability
8. Recoverability
9. Respectful permission handling
10. Invisible engineering

## Architecture Rule

Architecture exists to protect product promises.

Every future ADR must state which promises it protects.
