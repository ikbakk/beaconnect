# ADR-012 — Security & Privacy Strategy

Status: Frozen

## Decision

Beaconnect uses privacy by design.

## Rules

- one-to-one access only in v1
- no public profiles
- no discoverability by phone/email search
- no hidden sharing
- no profile photos in v1
- no raw location exposed unless necessary
- no detailed history beyond 3 days
- no selective history editing
- shared deletion requires mutual approval

## Analytics

Avoid behavioral analytics in v1.

Do not collect relationship behavior analytics.
