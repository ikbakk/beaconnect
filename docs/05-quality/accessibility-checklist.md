# Accessibility Checklist

## Text

- [ ] Supports large fonts
- [ ] No critical text truncation
- [ ] Contrast meets accessibility standards
- [ ] Copy is understandable without technical knowledge

## Controls

- [ ] Minimum 48dp touch targets
- [ ] TalkBack labels are meaningful
- [ ] Buttons describe outcome
- [ ] No color-only meaning

## Motion

- [ ] Reduced motion respected
- [ ] No looping decorative animations
- [ ] No flashing

## Widget

- [ ] Readable at glance
- [ ] High contrast
- [ ] TalkBack-friendly
- [ ] Action labels clear
